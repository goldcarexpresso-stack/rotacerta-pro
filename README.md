# RotaCerta Pro — Mobile Ready 1.2

Projeto preparado para trabalhar **pelo celular**: baixe o ZIP, envie para o GitHub e conecte na Vercel.

## O que tem nesta versão

- React + Vite
- Estrutura profissional em `src/`
- PWA instalável
- Firebase preparado via `.env.example`
- Login demo
- Dashboard
- Rotas
- Cadastro de pacotes com identificador alfanumérico
- Fotos dos pacotes e etiquetas
- OCR inicial com Tesseract
- Importação/exportação CSV
- Mapa da carga com foto do veículo
- Marcação de pacote na foto
- Google Maps
- Plano gratuito com 50 pacotes
- Premium demo
- Idiomas PT/EN/ES

## Como subir pelo celular

1. Baixe este ZIP.
2. Abra o GitHub no navegador do celular.
3. Crie um repositório chamado `rotacerta-pro`.
4. Envie os arquivos do ZIP para o repositório.
5. Abra a Vercel pelo navegador.
6. Clique em **New Project** e importe o repositório do GitHub.
7. Deploy.

## Rodar localmente em editor online

Use Replit, StackBlitz ou Firebase Studio. Depois rode:

```bash
npm install
npm run dev
```

## Firebase

Crie um projeto no Firebase e copie suas chaves para um arquivo `.env`, usando `.env.example` como modelo.

Sem Firebase, o app funciona em modo local/demo usando `localStorage`.
